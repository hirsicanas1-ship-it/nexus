# NEXUS - Adaptive Learning MVP

An AI-powered personal adaptive learning system based on metacognition, memory tracking, and intelligent practice.

## Features

- **Interactive Learning Demo** - Click on highlighted terms to see definitions
- **Metacognition-based** - Helps learners understand how they learn best
- **Memory Tracking** - Improves long-term retention through intelligent practice
- **Dark Theme UI** - Modern, easy-on-the-eyes interface with blue and purple accents

## Getting Started

Open `index.html` in your browser to start using the adaptive learning system.

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
